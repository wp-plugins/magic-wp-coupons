== Description ==

= What is Magic WP Coupons Plugin? =
DV Magic coupons is a WordPress based plugin which can magically turn your WordPress theme into a coupons site. It will allow you to add coupons and define stores before you add coupons.Dramatically adds coupon snippets into your site and and allows users of your site to browse, search and use coupon codes.

= Features =
1. Easy to install
2. One time affiliate link addition for each store
3. Stores Logos or thumbnails support
4. Comes with a decent look
5. Likes / Dislikes system
6. click counter system
7. External CSS support
8. Compatible with 99% WordPress templates available

== Installation ==

1. Upload `DV Magic Coupons` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all, your site will be transformed into coupon site

= How to use this plugin =
1. Firstly go to left menu in your wp-admin control panel and then click on **stores** , now you can add or edit stores here. Assign featured image to stores so that will be shown against coupon posts for that stores
2. Now you can add your coupons as normal posts, there will be custom meta box for you to add coupon details and assign to store etc.

== Frequently asked questions ==

= Will DV Magic Coupons transform my site into a coupon site? =

Yes! This will automatically transform your site into a coupon site, you just need to install DV Magic Coupons plugin and then activate it. It will do the magic.

= Is DV Magic Coupons compatible to all type of themes? =

Yes, any theme can be transformed into coupon theme with DV Magic theme however the only limit is the theme should be blog style theme.

= Can I customize the theme to change the look and feel with DV Magic Coupons? =

Yes, DV Magic Coupons allow you to use your own CSS. You can visit the DV Magic Coupon setting page in your admin panel to allow external CSS. It can pick up styles from your theme in use.
